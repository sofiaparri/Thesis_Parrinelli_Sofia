       README
--------------------

Die mex Datei ITAPolhemus.mex32 benötigt folgende zusätzliche Dateien:

- ITADeviceDrivers.dll
- sämtliche VistaCoreLibs release DLLs

Zum Initialisieren werden folgende Dateien benötigt

- eine Konfigurationsdatei, als Vorlage kann die PolhemusTracker.ini benutzt werde

Der Tracker benötigt zum initialisieren ein wenig Zeit. Manchmal klappt es trotzdem nicht.
Sollte ein erneuter Versuch den Tracker wieder nicht zum Laufen bringen, kann ein Neustart
des Polhemus Besserung bringen.
